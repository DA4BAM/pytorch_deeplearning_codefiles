# 10일 만에 배우는 파이토치 딥러닝 기초 – 코드 저장소

이 저장소는 책 *10일 만에 배우는 파이토치 딥러닝 기초*에서 사용한 모든 코드와 예제를 정리한 공간입니다.  
각 장(chapter)별로 폴더를 구성하여, 학습 흐름에 맞춰 실습 파일을 찾을 수 있도록 구성했습니다.

## 사용 방법

- 학습하려는 장(n일차) 파일 `.ipynb` 파일을 열어 내용을 확인합니다.  
- Google Colab 또는 로컬 환경(PyTorch 설치 필요)에서 실행할 수 있습니다.  

---

## 데이터 안내

- 실습용 데이터는 깃허브 저장소에는 data를 단축된 url로 제공합니다. 
- MNIST, FASHION-MNIST, CIFAR-10 등 일부 데이터는 코드를 실행하면 자동으로 다운로드됩니다.
